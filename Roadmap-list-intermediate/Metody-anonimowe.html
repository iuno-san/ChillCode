Metody Anonimowe W Języku C#

Metody anonimowe w języku C# są to metody, które nie posiadają nazwy i są definiowane bezpośrednio w miejscu ich użycia. Są one często używane do tworzenia delegatów lub jako parametry do innych metod.

Można je tworzyć za pomocą składni delegate lub za pomocą składni lambada. Przykład metody anonimowej za pomocą składni delegate: 

								
delegate void MyDelegate(int x);

MyDelegate myDelegate = delegate(int x) { Console.WriteLine(x); };

Przykład metody anonimowej za pomocą składni lambda: 

								
Func<int, int> myDelegate = x => x * x;

przykład pokazujący jak użyć metody anonimowej do sortowania listy liczb całkowitych: 

								
using System;
using System.Collections.Generic;

class Program
{
		static void Main(string[] args)
		{
				List<int> numbers = new List<int> { 9, 4, 2, 7, 5, 3 };


				// Użycie metody anonimowej za pomocą składni delegate do sortowania listy liczb rosnąco
				numbers.Sort(delegate(int x, int y) { return x.CompareTo(y); });


				// Użycie metody anonimowej za pomocą składni lambda do sortowania listy liczb malejąco
				numbers.Sort((x, y) => y.CompareTo(x));


				foreach (int number in numbers)
				{
						Console.WriteLine(number);
				}
		}
}


//output:

9
7
5
4
3
2
Uwaga

Metody anonimowe są często używane w programowaniu zdarzeniowym, takim jak reakcja na przyciski lub inne elementy interfejsu użytkownika, a także w przypadku programowania funkcyjnego, takim jak kompozycja funkcji lub przetwarzanie kolekcji danych

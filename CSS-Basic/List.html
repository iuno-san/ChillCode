Listy

Listy w CSS to elementy HTML, które pozwalają na prezentację informacji w postaci uporządkowanej lub nieuporządkowanej listy punktów. W CSS można zastosować wiele właściwości, takich jak typ punktora, położenie punktora, odstępy i marginesy, aby dostosować wygląd i prezentację list.

Nieuporządkowana lista Użyj tagu Użyj tagu ul do utworzenia nieuporządkowanej listy 

										
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
  • Element 1
  • Element 2
  • Element 3

Uporządkowana lista Użyj tagu ol do utworzenia uporządkowanej listy 

										
<ol style="list-style-type: upper-roman;">
  <li>Element 1</li>
  <li>Element 2</li>
  <li>Element 3</li>
</ol>
  1. Element 1
  2. Element 2
  3. Element 3

Zagnieżdżona lista Aby utworzyć zagnieżdżoną listę użyj tagu ul lub ol wewnątrz elementu listy. 

										
<ul>
  <li>Element 1
	<ul>
	  <li>podElement 1</li>
	  <li>podElement 2</li>
	</ul>
  </li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
  • Item 1
    • podElement 1
    • podElement 2
  • Element 2
  • Element 3

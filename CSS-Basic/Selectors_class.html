Selektor klasy

Selektor klasy pozwala na wybieranie elementów HTML o określonej klasie. Klasa jest atrybutem HTML, który można nadać dowolnemu elementowi i służy do grupowania elementów w celu łatwiejszego stylowania. Selektor klasy jest identyfikowany kropką przed nazwą klasy .nazwaklasy.

W poniższym przykładzie selektor klasy ".example" wybierze wszystkie elementy z klasą "example" i nada im kolor niebieski oraz rozmiar czcionki 16px. Aby użyć tego selektora klasy, należy nadać atrybut "class" odpowiedniemu elementowi HTML:

CSS

								
.example {

  color: blue;

  font-size: 16px;

}

HTML

										
<p class="example">Tekst z selektorem klasy</p>

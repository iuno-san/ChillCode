Wysokość i Szerokość

W CSS, szerokość i wysokość elementu można ustawić za pomocą właściwości "width" i "height". Te właściwości określają odpowiednio szerokość i wysokość elementu w pikselach lub jednostkach procentowych w stosunku do rodzica.

Wartości width i height można określić w różnych jednostkach, takich jak:

  • px - określa liczbę pikseli, czyli jednostek na ekranie, jakie będą zajmowane przez element
  • % - określają szerokość i wysokość elementu jako procent dostępnej przestrzeni.
  • cm - centymetry są jednostką długości, która mierzy rzeczywistą długość na ekranie.
  • in - cale są jednostką długości, która mierzy rzeczywistą długość na ekranie.
  • em - jednostka względna, która odnosi się do rozmiaru czcionki elementu nadrzędnego.

  Ten element ma 100% szerokości oraz 120px wysokości

width Określa szerokość elementu.

										
.element {

  width: 500px;

}

height Określa wysokość elementu.

										
.element {

  height: 300px;

}

min-width Określa minimalną szerokość elementu.

										
.element {

  min-width: 200px;

}

max-width Określa maksymalną szerokość elementu.

										
.element {

  max-width: 800px;

}

min-height Określa minimalną wysokość elementu.

										
.element {

  min-height: 100px;

}

max-height Określa maksymalną wysokość elementu.

										
.element {

  max-height: 1000px;

}

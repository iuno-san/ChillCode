Dopełnienie

Padding to właściwość, która służy do dodawania marginesu wewnętrznego do elementu HTML. Margines wewnętrzny jest to przestrzeń pomiędzy zawartością elementu a jego obrysem. Właściwość padding może być ustawiona dla każdej ze stron elementu lub dla wszystkich jednocześnie. Może być również ustawiona na stałą wartość, np. 20 pikseli, lub procentowo w stosunku do wielkości elementu rodzica.

Ustawienie padding dla wszystkich stron na 65px. 

								
.element {
  padding: 65px;
}
Ten element ma padding 65px.

Ustawienie padding dla każdej ze stron na różną wartość: 

						
.element {
  padding-top: 10px;
  padding-right: 15px;
  padding-bottom: 20px;
  padding-left: 25px;

}
Ten element ma wiele dopełnień.

Ustawienie padding po lewej stronie elementu na 10% jego szerokości: 

								
.element {
  padding-left: 10%;
}
Ten element ma 10% padding po lewej stronie elementu.
Wyjaśnienie
  • padding-top: 20px; - dodaje 20 pikseli marginesu na górnej stronie elementu.
  • padding-right: 10%; - dodaje 10% szerokości elementu rodzica jako margines po prawej stron
Notatka

Padding jest narzędziem, które pozwala na lepszą kontrolę nad wyglądem naszej strony internetowej i dostosowanie go do naszych potrzeb i wymagań.

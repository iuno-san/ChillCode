Właściwość float

Właściwość "float" w CSS służy do ustawiania elementów na stronie w taki sposób, że są one wyświetlane na bokach strony, zamiast w swoim normalnym miejscu, jakby były ustawione jako "block". Wartości tej właściwości to "left", "right", "none" (domyślna) i "inherit". Właściwość ta jest często używana do układania elementów na stronie, takich jak obrazy i tekst.

Można ustawić element jako float: left lub float: right aby ustawić go po lewej lub prawej stronie, zostawiając tekst i inne elementy wokół niego w swoim normalnym miejscu. Jeśli chcesz, aby element nie był ustawiony jako "float". 

								
img {

  float: left;
  margin-right: 10px;
}

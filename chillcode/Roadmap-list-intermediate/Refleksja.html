Refleksja W Języku C#

Refleksja (ang. reflection) to mechanizm dostępny w języku C#, pozwalający na dynamiczne odczytywanie informacji o typach, ich składowych i wywoływanie metod. Umożliwia on programowi na "odpytywanie" obiektów o ich typy i właściwości w czasie wykonywania.

Refleksja jest szczególnie przydatna przy tworzeniu uniwersalnych narzędzi, które muszą działać z różnymi typami danych bez ich wcześniejszego znajomości.

Refleksja w języku C# umożliwia programiście m.in.:

  • Odczytywanie informacji o typach i ich składowych (np. pola, metody, konstruktory)
  • Tworzenie obiektów na podstawie informacji o typie
  • Dynamiczne wywoływanie metod i uzyskiwanie informacji o ich parametrach
  • Uzyskiwanie informacji o atrybutach (np. adnotacjach) znajdujących się na typach i ich składowych
  • Dynamiczne ustawianie wartości pól
  • Tworzenie dynamicznych proxy (np. do celów testowych)
  • Utworzenie narzędzi automatyzujących generowanie kodu, używających refleksji.

Przykład użycia refleksji w języku C#, w którym tworzymy instancję obiektu klasy "Example" za pomocą metody "Activator.CreateInstance()", a następnie uzyskujemy informacje o typie tej klasy za pomocą klasy "Type": 

								
using System;
using System.Reflection;

class Example
{
		public int a;
		public string b;

		public Example(int a, string b)
		{
				this.a = a;
				this.b = b;
		}
}


class Program
{
		static void Main(string[] args)
		{
				// Tworzenie instancji obiektu klasy Example
				var example = (Example)Activator.CreateInstance(typeof(Example), 5, "Hello");


				// Uzyskanie informacji o typie obiektu example
				Type type = example.GetType();


				// Wypisanie nazwy typu
				Console.WriteLine("Type name: " + type.Name);


				// Wypisanie liczby pol klasy
				Console.WriteLine("Properties count: " + type.GetProperties().Length);


				// Wypisanie liczby metod klasy
				Console.WriteLine("Methods count: " + type.GetMethods().Length);
		}
}


//output:

Type name: Example
Properties count: 0
Methods count: 4

Wyświetlanie Metadanych

Wyświetlanie metadanych to proces odczytywania informacji o typach, ich składowych (np. polach, metodach, konstruktorach) i atrybutach (np. adnotacjach) znajdujących się w programie w czasie jego wykonywania.

Metoda Type.GetType() 

								
Type myType = Type.GetType("MyNamespace.MyClass");

Metoda typeof() 

								
Type myType = typeof(MyClass);

Metoda object.GetType() 

								
MyClass myObject = new MyClass();

Type myType = myObject.GetType();

Metoda Type.GetProperties() 

								
Type myType = typeof(MyClass);

foreach (PropertyInfo property in myType.GetProperties())
{
		Console.WriteLine("Nazwa właściwości: " + property.Name);
}

Metoda Type.GetMethods() 

								
Type myType = typeof(MyClass);

foreach (MethodInfo method in myType.GetMethods())
{
		Console.WriteLine("Nazwa metody: " + method.Name);
}

Metoda Type.GetFields() 

								
Type myType = typeof(MyClass);

foreach (FieldInfo field in myType.GetFields())
{
		Console.WriteLine("Nazwa pola: " + field.Name);
}

Metoda Type.GetConstructors() 

								
Type myType = typeof(MyClass);

foreach (ConstructorInfo constructor in myType.GetConstructors())
{
		Console.WriteLine("Nazwa konstruktora: " + constructor.Name);
}

Metoda Type.GetAttributes() 

								
Type myType = typeof(MyClass);

object[] attributes = myType.GetCustomAttributes(typeof(MyAttribute), false);

foreach (MyAttribute attribute in attributes)
{
		Console.WriteLine("Nazwa atrybutu: " + attribute.Name);
}
Uwaga

Oczywiście, te przykłady są najprostszymi i najogólniejszymi przykładami użycia tych metod, istnieją również inne metody i opcje, które pozwalają na bardziej zaawansowane zastosowanie refleksji.

Instancja Klasy W Języku C#

Instancja klasy to konkretny obiekt reprezentujący klasę, który jest tworzony na podstawie tej klasy. Tworzenie instancji klasy polega na wykonaniu operacji "new", która tworzy nowy obiekt w pamięci i przypisuje mu odpowiednie wartości początkowe.

W języku C#, możemy utworzyć instancję klasy za pomocą operatora "new" i wywołania konstruktora, np: 

								
Example example = new Example(5, "Hello");

Refleksja pozwala na tworzenie instancji klasy bez wiedzy o jej typie przed czasem kompilacji, za pomocą metody Activator.CreateInstance() 

								
Example example = (Example)Activator.CreateInstance(typeof(Example), 5, "Hello");

przykład użycia instancji klasy w języku C#, w którym tworzymy kilka instancji klasy "Person" i używamy ich do przechowywania informacji o osobach: 

								
using System;

class Person
{
		public string Name { get; set; }
		public int Age { get; set; }

		public Person(string name, int age)
		{
				Name = name;
				Age = age;
		}
}


class Program
{
		static void Main(string[] args)
		{
				// Tworzenie instancji obiektów klasy Person
				Person person1 = new Person("darek", 22);
				Person person2 = new Person("Sara", 15);
				Person person3 = new Person("Janusz", 35);


				// Wypisanie informacji o osobach
				Console.WriteLine("Name: " + person1.Name + ", Age: " + person1.Age);
				Console.WriteLine("Name: " + person2.Name + ", Age: " + person2.Age);
				Console.WriteLine("Name: " + person3.Name + ", Age: " + person3.Age);
		}
}


//output:

Name: darek, Age: 22
Name: Sara, Age: 15
Name: Janusz, Age: 35

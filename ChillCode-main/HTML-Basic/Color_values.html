Wartości kolorów

W języku HTML i CSS istnieją 3 różne systemy wartości kolorów: RGB (czerwień, zieleń, niebieski), HEX (heksadecymalne) i HSL (odcień, nasycenie, jasność). Każdy z nich określa kolor przez różne składniki i jest używany w różnych sytuacjach.

Ustawienie koloru tła strony internetowej przy użyciu rgb 

							
<body style="background-color: rgb(128, 255, 0);">  <!-- kolor zielony -->

Ustawienie koloru tła strony internetowej przy użyciu hsla 

							
<body style="background-color: hsla(9, 100%, 64%, 0.6);">  <!-- kolor pomarańczowy -->

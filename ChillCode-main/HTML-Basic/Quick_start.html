Szybki start w HTML

Cześć! HTML (Hypertext Markup Language) to podstawowy język używany do tworzenia stron internetowych. Oto kilka podstawowych kroków, które pozwolą Ci rozpocząć pracę z HTML:

Oto krótki przewodnik, jak zacząć pisać w HTML:

1. Otwórz edytor tekstu. Możesz użyć prostego edytora tekstu, takiego jak Notatnik lub bardziej zaawansowanego, takiego jak Visual Studio Code.

2. Utwórz nowy plik i zapisz go z rozszerzeniem .html.

3. Rozpocznij pisanie kodu HTML od tagu <html> który otwiera i zamyka wszystkie elementy HTML.

4. Po zakończeniu pisania kodu HTML, zapisz plik i otwórz go w przeglądarce internetowej, aby zobaczyć, jak wygląda Twoja strona.

Przykładowy kod HTML dla prostej strony

				
<!DOCTYPE html>
<html>
<head>
    <title>Moja pierwsza strona internetowa</title>
    <meta charset="UTF-8">
    <meta name="description" content="Opis strony.">
    <meta name="keywords" content="HTML, tutorial, strona internetowa">
    <link rel="stylesheet" href="styl.css">
</head>
<body>
<header>
    <h1>Moja pierwsza strona internetowa</h1>
</header>

<nav>
    <ul>
        <li><a href="#">Strona główna</a></li>
        <li><a href="#">O mnie</a></li>
        <li><a href="#">Kontakt</a></li>
    </ul>
</nav>

<main>
    <article>
        <h2>Tytuł artykułu</h2>
        <p>To jest mój pierwszy artykuł na stronie internetowej.</p>
        <img src="flower.jpg" alt="Opis obrazu">
    </article>
</main>

<footer>
    <p>© 2023 Moja pierwsza strona internetowa</p>
</footer>
</body>
</html>

Uruchamianie kodu

Wystarczy kliknąć przycisk "Uruchom kod" i zobaczyć wynik działania programu 

									#include <iostream>

using namespace std;

int main()
{
  int a = 6;
  int b = 5;

  int suma = a + b;

  cout << "Wynik to: " << suma;

  return 0;
}
Wynik to: 11

Podpowiedzi

Najedź na podświetlone słowo kluczowe, które cię interesuje i zobacz prostą definicje

Ten program w języku C++ dodaje dwie liczby pry użyciu int i wyświetla ich sumę na ekranie konsoli. Wykorzystuje preprocesor #include <iostream> , który umożliwia korzystanie ze strumieni wejścia i wyjścia oraz definiuje zmienne a i b, dodaje je i przypisuje wynik do zmiennej suma. Wynik dodawania jest wyświetlany na ekranie konsoli za pomocą funkcji cout . Program kończy się zwracając wartość 0, oznaczając, że wykonanie programu zakończyło się poprawnie.

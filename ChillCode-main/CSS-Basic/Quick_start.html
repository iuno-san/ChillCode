Szybki start

CSS (Cascading Style Sheets) to język stosowany do opisu wyglądu i formatowania dokumentów HTML, XML i innych podobnych struktur dokumentów. CSS umożliwia oddzielenie struktury dokumentu od jego prezentacji, co ułatwia zarządzanie i zmienianie stylów strony.

Oto krótki przewodnik, jak zacząć pisać w CSS:

1. Aby użyć CSS, musisz najpierw utworzyć plik HTML. Można to zrobić przy użyciu edytora tekstu, takiego jak Notepad lub bardziej zaawansowanego, np Visual Studio Code

2. Następnie musisz utworzyć plik CSS. Można to zrobić w osobnym pliku lub wewnątrz pliku HTML przy użyciu tagu <style>.

3. Wybierz elementy HTML, na które chcesz zastosować styl, możesz wybrać dowolny element HTML, takie jak nagłówki, paragrafy, linki, obrazy itp.

4. W pliku CSS możesz zdefiniować styl dla wybranych elementów HTML, takie jak kolor, czcionkę, marginesy, obramowanie itp.

5. Aby zastosować style z pliku CSS na stronie HTML, musisz połączyć plik CSS z plikiem HTML. Można to zrobić przy użyciu tagu <link> w sekcji nagłówka strony HTML.

6. Po zdefiniowaniu stylów i połączeniu plików, sprawdź, jak wygląda strona. Możesz wprowadzać zmiany w pliku CSS i obserwować, jak wpływają one na stronę.

Przykład zastosowania CSS

								
<!DOCTYPE html>
<html>
<head>

<title>Zmiana koloru tekstu i tła</title>

<style>

/* Zmiana koloru tekstu na niebieski */
p {
  color: blue;
}

/* Zmiana koloru tła na średni fioletowy */
body {
  background-color: mediumpurple;
}

</style>

</head>
  <body>
    <p>Jestem niebieskim tekstem</p>
  </body>
</html>

Jestem niebieskim tekstem

Gratulacje

CSS jest bardzo elastycznym językiem, więc eksperymentuj z różnymi stylami i efektami, aby uzyskać pożądany wygląd strony.. Powodzenia!

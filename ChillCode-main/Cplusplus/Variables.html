Zmienne

W C++, zmienne są deklarowane, aby przechowywać dane. Każda zmienna ma swój typ, który określa, jakiego rodzaju dane będą przechowywane, np. liczby całkowite, liczby zmiennoprzecinkowe, znaki, itp.

Zmienne w C++ są definiowane przez podanie nazwy zmiennej oraz typu danych, który jest przechowywany w zmiennej. Na przykład, aby zdefiniować zmienną całkowitą o nazwie "x", możemy napisać: 

								
int x;   // zmienna typu int przechowująca liczby całkowite

Możemy również zainicjować zmienną podczas jej definiowania 

 								
int y = 10;   // zmienna typu int zainicjowana wartością 10

Możemy również użyć wartości innej zmiennej, aby przypisać wartość do innej zmiennej 

								
#include <iostream>

using namespace std;

int main()
{
    int a = 10;
    int b = 5;
    int suma = a + b;   // zmienna "suma" przechowuje sumę wartości zmiennych "a" i "b"
    cout << "Suma: " << suma << endl;  // wyświetla wynik na ekranie
    return 0;
}
Suma: 15

W C++ można również definiować stałe przy użyciu słowa kluczowego const Stałe nie mogą być zmieniane po zdefiniowaniu. 

								
const int PI = 3.14;
Notatka

Zmienne w C++ są deklarowane, aby przechowywać dane. Każda zmienna ma swój typ i można ją zainicjować wartością podczas deklaracji. Zmienne są również wykorzystywane do przetwarzania danych w programach.

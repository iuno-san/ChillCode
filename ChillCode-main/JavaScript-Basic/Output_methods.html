Metody wyjścia

W języku JavaScript istnieją kilka sposobów wyświetlenia wyniku (outputu) na ekranie. Najczęściej używanym sposobem jest funkcja console.log(). Funkcja ta pozwala wyświetlić tekst lub wartość zmiennej na konsoli przeglądarki internetowej.

Przykład użycia funkcji console.log() która wyświetla tekst lub wartość zmiennej na stronie HTML 

								
console.log("Hello, World!");
Hello, World!

Użycie funkcji document.write() która wyświetla tekst lub wartość zmiennej na stronie HTML. 

								
document.write("Hello, World!");

Innym sposobem jest użycie metody innerHTML elementu HTML, aby wyświetlić tekst lub wartość zmiennej w elemencie HTML. 

								
document.getElementById("output").innerHTML = "Hello, World!";
Notatka

Pamiętaj, że powyższe przykłady są tylko wstępem do tego, co można zrobić z wynikiem (outputem) w języku JavaScript. Można zrobić wiele więcej, takich jak formatowanie tekstu, wyświetlanie wartości zmiennych i inne. Wszystko zależy od potrzeb i wymagań projektu.

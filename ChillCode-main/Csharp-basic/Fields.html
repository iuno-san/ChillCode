Pola

Pola w języku C# to elementy składowe klasy lub struktury, które przechowują dane. Są one podobne do zmiennych, ale są one częścią klasy lub struktury i są dostępne dla wszystkich metod w klasie lub strukturze.

Oto przykład deklaracji pola w języku C# przy użyciu modyfikatora public 

								
using System;

public class MojaKlasa

{
    public int _mojePole;
}
Wyjaśnienie

W tym przykładzie deklarowane jest pole _mojePole typu int w klasie MojaKlasa<. Pole to jest publiczne, co oznacza, że jest dostępne dla wszystkich metod w klasie MojaKlasa oraz dla innych klas.

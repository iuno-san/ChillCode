Atrybuty tabeli

Atrybuty tabeli to opcjonalne parametry, które można dodać do elementu <table> lub do innych elementów wewnątrz tabeli, aby zmienić wygląd lub zachowanie tabeli, aby była ona czytelna i przejrzysta dla użytkownika.

Najważniejsze atrybuty tabeli w HTML to:

border cellpadding cellspacing width height bgcolor summary align valign colspan rowspan caption 

								
<table border="1" cellpadding="5" cellspacing="0" width="600" bgcolor="#f2f2f2" align="center" valign="middle" summary="Wyniki meczów">

    <caption>Wyniki meczów</caption>
    <tr>
        <th>Data</th>
        <th>Gospodarz</th>
        <th>Gość</th>
        <th>Wynik</th>
    </tr>
    <tr>
        <td>02.03.2023</td>
        <td>Real Madryt</td>
        <td>Barcelona</td>
        <td>3:1</td>
    </tr>
    <tr>
        <td>04.03.2023</td>
        <td>Liverpool</td>
        <td>Manchester United</td>
        <td>2:0</td>
    </tr>
    <tr>
        <td>05.03.2023</td>
        <td>Paris Saint-Germain</td>
        <td>Marseille</td>
        <td>2:2</td>
    </tr>

</table>

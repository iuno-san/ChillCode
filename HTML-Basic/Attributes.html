Atrybuty

Atrybuty HTML to specyficzne dla danego elementu właściwości, które pozwalają na określenie jego zachowania lub wyglądu. Służą do określania różnych właściwości elementów HTML. Mogą one mieć różne cele, takie jak określenie linków, zawartości, stylów, zachowania i wielu innych. Atrybuty są zawsze zdefiniowane jako pary "nazwa/wartość" i są zawarte w tagu HTML.

href - definiuje adres URL 

											
<a href="https://www.Joke.com">Link do strony</a>

src - definiuje źródło obrazka 

											
<img src="image.jpg" alt="Opis obrazu">

class - przypisuje klasę do elementu 

											
<p class="text-blue">Tekst niebieski</p>

id - nadaje identyfikator elementowi 

											
<div id="header">Zawartość nagłówka</div>

style - określa style CSS 

										
<p style="color: blue;">Tekst niebieski</p>

width i height - szerokość i wysokość 

										
<img src="image.jpg" width="300" height="200">

title - definiuje tekst 

										
<a href="Joke.com" title="Strona">Link</a>

lang - definiuje język dokumentu 

										
<html lang="pl">
Notatka

Jak widzisz, atrybuty HTML pozwalają na różnego rodzaju konfiguracje i personalizacje elementów, co jest bardzo przydatne w tworzeniu różnorodnych stron internetowych.

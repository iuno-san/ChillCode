Operatory logiczne

Operatory logiczne w języku JavaScript pozwalają na kombinowanie wartości logicznych i zwracanie wartości logicznej true lub false. Oto kilka przykładów operatorów logicznych:

&& Jeśli oba operandy są równe true, wynik jest równy true. W przeciwnym razie wynik jest równy false.

										
let x = true;
let y = false;

console.log(x && y);
false

|| Jeśli co najmniej jeden z operandów jest równy true, wynik jest równy true. W przeciwnym razie wynik jest równy false.

										
let x = true;
let y = false;

console.log(x || y);
true

! Zmienia wartość logiczną operanda. true staje się false, a false staje się true.

								
let x = true;

console.log(!x); // false
false
Notatka

Operatory logiczne są często używane w warunkach, instrukcjach if i switch, a także w pętlach, aby kontrolować, kiedy wykonywać konkretne czynności.
